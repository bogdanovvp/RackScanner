Windows:
- install miniconda
- in the Anaconda prompt:
- clone https://github.com/michalkahle/RackScanner.git
- cd to install directory, then run: 
	conda env create -f conda_env.yaml
	or
	conda create --name RackScanner --file conda_env_win32.txt




instalation of AVA6 driver in Windows 8.1:
- restart the system with driver signature enforcement turned off!
- run the original installer from the CD and install TWAIN and W?? driver (might not be necessary)
- from Device Manager start the driver update process
- select driver manually -> have a disk
- select the installer directory Avision Scanner\Driver\TWAIN
- you should get warning about insalling unsigned drivers. Proceed.
- unplug, plug the USB cable. The scanner should work now.
- only 32 bit TWAIN is supported!

install twain driver:
- go to http://www.lfd.uci.edu/~gohlke/pythonlibs/#twainmodule
- download and install twain‑1.0.4‑cp27‑cp27m‑win32.whl


install libdmtx and pydmtx:
- download source and build libdmtx.dll
- go to https://github.com/NaturalHistoryMuseum/dmtx-wrappers
- download wheel and install it
- install "Visual C++ Redistributable Packages for Visual Studio 2013" as explained on the page
